# Notebooks

Use this folder for exploratory review notebooks after the scripted pipeline has produced checkpoint `.h5ad` files.

Recommended first notebook:

1. Load `results/scanpy/03_clustered_umap.h5ad`.
2. Inspect QC distributions by batch and cluster.
3. Review top markers from `results/scanpy/marker_genes_by_leiden.csv`.
4. Adjust `config/config.yaml` thresholds or marker tables if needed.
